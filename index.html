ASML AI Board - Mistral Demo

About This Project

This repository contains a Streamlit application for ASML AI Board with search functionality.

The application implements Jira user story MP-3: "Add a Search Button to the python file".

View the Code

You can view the main application code here:

View asml_ai_board.py

How to Run Locally

To run this Streamlit application locally:

git clone https://github.com/Venkat8870/mistral-demo.git
cd mistral-demo
pip install streamlit
streamlit run asml_ai_board.py

Deploy to Streamlit Cloud

To deploy this application to Streamlit Cloud:

  1. Go to Streamlit Cloud
  2. Click "New app" and connect your GitHub account
  3. Select this repository (Venkat8870/mistral-demo)
  4. Set the main file path to: asml_ai_board.py
  5. Click "Deploy"

Application Features

  • ASML AI Board title display
  • Search input field
  • Search button with validation
  • Basic search display functionality
