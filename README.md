# ASML AI Board - Mistral Demo

This repository contains a Streamlit application for ASML AI Board with search functionality.

## 📁 Files

- `asml_ai_board.py` - Main Streamlit application
- `index.html` - GitHub Pages landing page

## 🚀 How to Run

```bash
git clone https://github.com/Venkat8870/mistral-demo.git
cd mistral-demo
pip install streamlit
streamlit run asml_ai_board.py
```

## 🌐 GitHub Pages

The GitHub Pages site should be available at: https://venkat8870.github.io/mistral-demo/

## 📝 About

This application implements Jira user story MP-3: "Add a Search Button to the python file".

Features:
- ASML AI Board title display
- Search input field
- Search button with validation
- Basic search display functionality
